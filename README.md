# Gameboy Advance Emulator  
Open source gba emulator in C++. Still a work in progress.  

## Progress  
Done:
* Bios loading
* Game rom loading

Not yet supported:  
* GamePak 
* Video
* Audio 
* Cpu interrupts
* Cpu timings
* Dma
* Keypad inputs
* Serial communication
* Timers

In progress:
* Cpu instruction sets (~20% done)
* Gba memory mapping
